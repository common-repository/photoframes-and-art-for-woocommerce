== Description ==

Using this plugin, you can build a platform on your website where visitors may design their fine art and purchase them.

Administrator can manage variety of components like default Backgrounds, Print types, Paper types, Sizes, Frame Materials, Fixture, Glass, Window mount, Substrate. All of these components will appear on the selected page so that users may use them in their own fine art.

Kindly let us know your feedback or comments to add more features in this plugin.

== Installation ==

1. Download and save the *Photoframes and art for WooCommerce* plugin to your hard disk.
2. Login to your WordPress and go to the Add Plugins page.
3. Click Upload Plugin button to upload the zip.
4. Click Install Now to install and activate the plugin.

== Features ==

- Easy for an administrator to utilise to control settings

- Make your own exquisite art

- Manage framing choices

- Upload own layout

- Convenient for consumers to utilise with make an order

- Checkout in 3 simple steps
	- Upload an image
	- Choose print type and/or framing
	- Checkout

== Frequently Asked Questions ==

= Does this plugin function without "WooCommerce"? =
No, WooCommerce must be installed and enabled in order for this plugin to work.

= How to set fine art editor page? =
The page selection option is located in the plugin's general settings. https://prnt.sc/EhcWQZl-3PBd

= How users can order their fine art? =
The ordering process for this plugin uses the WooCommerce checkout. Users can simply design the art and checkout.
